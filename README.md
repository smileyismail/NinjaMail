## Tech Stack

React, Tailwind CSS

## Run Locally

Clone the project

```bash
  git clone https://github.com/smileyismail/Movies-Hive
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
